Undergraduate Dissertation Project {

"A Comparative Analysis of CPython and Mojo for Performance and Usability."

Developed a comparative analysis of CPython and its superset Mojo, by implementing comparable algorithm implementations and testing key benchmarks. Investigated the different approaches taken in language and compiler design, and their impacts on performance and usability.

Algorithm Implementations Compared:

  • Matrix Multiplication
  • Quick Sort
  • A* Pathfinding
  • Conway's Game of Life

Through utilising Mojo's more explicit control over low system resources, excecution time of algorithms was increased by up to 5x over standard CPython.

}

The Multi-Agent System Framework provides the building blocks to create a multi-agent system. Enables the creation of multiple agents that interact with each other and a shared environment. Agents act independently, and run asynchronously. Can be used as a foundation for a number of MAS projects.
